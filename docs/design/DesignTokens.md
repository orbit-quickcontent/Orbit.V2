# Orbit Design Tokens

Comprehensive design token definitions extracted directly from `orbit-marketplace/dashboard-web-app/src/app/globals.css` and `tailwind.config.ts`.

## 1. Color Tokens
- `Space Navy` (Background): `#000000`
- `Space Navy Light` (Card / Modal Surface): `#0A0A0A`
- `Space Navy Lighter` (Secondary Surface): `#111111`
- `Orbit Cyan` (Primary Accent / Glow): `#00BFFF`
- `Orbit Purple` (Secondary Accent / Glow): `#A020F0`
- `Orbit Border`: `#1A1A2E`
- `Orbit Muted`: `#7EB8E0`
- `Orbit Destructive`: `#FF4444`

## 2. Gradient Tokens
- `Orbit Gradient Primary`: `linear-gradient(135deg, #00BFFF 0%, #A020F0 100%)`
- `Orbit Glow Shadow`: `0 0 20px rgba(0, 191, 255, 0.25), 0 0 60px rgba(160, 32, 240, 0.1)`

## 3. Glassmorphism Tokens
- `Orbit Card Glass`: `background: rgba(10, 10, 10, 0.85); backdrop-filter: blur(12px); border: 1px solid rgba(0, 191, 255, 0.12);`
- `Orbit Card Strong Glass`: `background: rgba(10, 10, 10, 0.95); backdrop-filter: blur(16px); border: 1px solid rgba(0, 191, 255, 0.2);`
- `Orbit Nav Pill Glass`: `background: rgba(5, 5, 5, 0.75); backdrop-filter: blur(16px); border: 1px solid rgba(0, 191, 255, 0.12); border-radius: 28px;`

## 4. Radius Tokens
- `Radius Base`: `12px` (`0.75rem`)
