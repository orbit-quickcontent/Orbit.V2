|---|---|---|---|---|

| `--card` | `#0A0A0A` | `hsl(0 0% 4%)` | `rgb(10, 10, 10)` | Card and modal surface background |
| `--secondary` | `#111111` | `hsl(0 0% 6.7%)` | `rgb(17, 17, 17)` | Secondary container / input background |
| `--primary` | `#00BFFF` | `hsl(195 100% 50%)` | `rgb(0, 191, 255)` | Orbit Cyan - Main action buttons, active tabs |

| `--border` | `#1A1A2E` | `hsl(240 28% 14%)` | `rgb(26, 26, 46)` | Divider lines, card outlines |
| `--muted-foreground` | `#7EB8E0` | `hsl(205 63% 68%)` | `rgb(126, 184, 224)` | Subtitle text, caption labels |
| `--destructive` | `#FF4444` | `hsl(0 100% 63%)` | `rgb(255, 68, 68)` | Error alerts, record buttons, sign-out actions |
