# Orbit Native Custom Component Specifications

Inventory of custom components to be built across Android Kotlin Compose and iOS SwiftUI:

1. `OrbitButton` / `OrbitGradientButton`
2. `OrbitGlassCard` / `OrbitCardStrong`
3. `OrbitInput` / `OrbitTextField`
4. `OrbitSearchBar`
5. `OrbitTopBar` / `OrbitHeader`
6. `OrbitBottomNavigation` (`orbit-nav-pill`)
7. `OrbitPackageCard`
8. `OrbitStatusTimeline` (5-stage live status)
9. `OrbitWalletCard`
10. `OrbitBookingCard`
11. `OrbitProfileCard`
12. `OrbitNotificationCard`
13. `OrbitDialogs`
14. `OrbitBottomSheet`
15. `OrbitLoader` / `OrbitSkeleton`
