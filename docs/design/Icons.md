# Orbit Iconography System

Mapping Lucide React web icons to native vector icons (Android Compose & iOS SF Symbols):

| Web Component Icon | Android Compose Icon | iOS SF Symbol | Purpose |
|---|---|---|---|
| `Video` / `Film` | `Icons.Default.PlayArrow` / `Movie` | `video.fill` | Shoot & Reel indicator |
| `Calendar` | `Icons.Default.DateRange` | `calendar` | Date picker slot |
| `MapPin` | `Icons.Default.LocationOn` | `mappin.and.ellipse` | Shoot location address |
| `CreditCard` / `Wallet` | `Icons.Default.AccountBalanceWallet` | `creditcard.fill` | Payouts & Balance |
| `User` | `Icons.Default.Person` | `person.fill` | Profile & Brand DNA |
| `CheckCircle` | `Icons.Default.CheckCircle` | `checkmark.circle.fill` | Status complete |
| `Bell` | `Icons.Default.Notifications` | `bell.fill` | Notifications sheet |
