# Orbit Spacing & Grid System Specifications

Standard 4px base grid system for Orbit native applications:

| Token Name | Size (dp / pt) | Usage |
|---|---|---|
| `Spacing-XS` | `4dp` | Internal chip padding, micro gaps |
| `Spacing-SM` | `8dp` | Button inner padding, icon text offset |
| `Spacing-MD` | `12dp` | Card vertical element spacing |
| `Spacing-LG` | `16dp` | Screen edge padding, standard card margins |
| `Spacing-XL` | `20dp` | GlassCard internal padding |
| `Spacing-2XL` | `24dp` | Section gap, modal padding |
| `Spacing-3XL` | `32dp` | Top hero section margin |
