# Orbit Button Specifications

## 1. OrbitGradientButton
- **Background**: Linear Gradient `#00BFFF` (Orbit Cyan) → `#A020F0` (Orbit Purple) at 135 deg angle.
- **Height**: `50dp` / `50pt`
- **Border Radius**: `12dp` / `12pt`
- **Typography**: `16sp`, Bold, `#FFFFFF`
- **Disabled State**: Dark Gray `#333333` gradient

## 2. OrbitSecondaryButton
- **Background**: `#111111`
- **Border**: `1px solid #1A1A2E`
- **Height**: `48dp` / `48pt`
- **Border Radius**: `12dp` / `12pt`
- **Typography**: `14sp`, Bold, `#FFFFFF`

## 3. OrbitDestructiveButton
- **Background**: `#FF4444`
- **Height**: `50dp` / `50pt`
- **Border Radius**: `12dp` / `12pt`
- **Typography**: `16sp`, Bold, `#FFFFFF`
