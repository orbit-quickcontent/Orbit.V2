# Orbit Card Specifications

## 1. OrbitGlassCard
- **Background**: `rgba(10, 10, 10, 0.85)` (`#0A0A0A` at 85% opacity)
- **Backdrop Blur**: `12px`
- **Border**: `1px solid #1A1A2E` (or `rgba(0, 191, 255, 0.12)`)
- **Border Radius**: `16dp` / `16pt`
- **Padding**: `20dp` / `20pt`

## 2. OrbitCardStrong
- **Background**: `rgba(10, 10, 10, 0.95)`
- **Backdrop Blur**: `16px`
- **Border**: `1px solid rgba(0, 191, 255, 0.2)`
- **Border Radius**: `16dp` / `16pt`

## 3. OrbitGlowCard
- **Shadow**: `0 0 20px rgba(0, 191, 255, 0.25), 0 0 60px rgba(160, 32, 240, 0.1)`
