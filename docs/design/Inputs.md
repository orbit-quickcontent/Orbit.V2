# Orbit Input Component Specifications

## 1. OrbitTextField
- **Background**: `#111111`
- **Focused Border**: `#00BFFF` (Orbit Cyan)
- **Unfocused Border**: `#1A1A2E`
- **Label Color**: `#7EB8E0` (Focused: `#00BFFF`)
- **Text Color**: `#FFFFFF`
- **Border Radius**: `10dp` / `10pt`

## 2. OrbitSearchBar
- **Background**: `#0A0A0A`
- **Border**: `1px solid #1A1A2E`
- **Icon**: Search / Magnifying Glass in `#00BFFF`
- **Border Radius**: `24dp` / `24pt`
