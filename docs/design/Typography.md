# Orbit Typography System Specifications

Extracted from Google Fonts configuration in `globals.css`:
- `Inter` (Sans Serif Base)
- `Space Grotesk` (Display & Headers)
- `Playfair Display` (Editorial Highlights)
- `Montserrat` / `Roboto` (Subtext & Badges)

## Font Scale & Weights

| Style Role | Font Family | Size (sp / pt) | Weight | Letter Spacing |

| Display Hero | `Orbit Brand / Space Grotesk` | `36sp` / `36pt` | ExtraBold / Black | `6sp` / `6pt` |
| Screen Title | `Space Grotesk` | `24sp` / `24pt` | Bold | `0` |
| Card Title | `Inter` | `20sp` / `20pt` | Bold | `0` |
| Body Main | `Inter` | `15sp` / `15pt` | Medium / Normal | `0` |
| Subtitle / Caption | `Inter` / `Montserrat` | `13sp` / `13pt` | Normal | `0` |
