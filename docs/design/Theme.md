# Orbit Theme System Specifications

Dark-first visual theme contract:
- **Theme Mode**: Enforced Dark Mode (`.dark` theme tokens)
- **Background**: Deep Space Black (`#000000`)
- **Accent Primary**: Orbit Cyan (`#00BFFF`)
- **Accent Secondary**: Orbit Purple (`#A020F0`)
- **Surface Elevation**: Layered glass cards (`#0A0A0A` at 0.85/0.95 opacity)
- **Border System**: 1px subtle glow outlines (`#1A1A2E` & `rgba(0, 191, 255, 0.12)`)
